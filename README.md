## 100tifi.co

A Single Page Aplication.

# DEMO
[https://100tifi.co](https://100tifi.co)

## Install Dependencies
```
npm install
```

## Develop mode
```
npm run dev
```

## Build Project
```
npm run build
```

## Compile CSS (Stylus)
```
npm run stylus
```

## Server 
```
npm run server
```
